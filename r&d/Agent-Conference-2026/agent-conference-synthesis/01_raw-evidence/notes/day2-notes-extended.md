# RAW THESIS RESIDUE

Agents should not be trusted.
Systems should be built so agents do not need to be trusted.

Code got safe because Git made failure cheap.
Data is still dangerous because failure is shared, slow, and catastrophic.

The agent layer is not the model.
The agent layer is schema, identity, policy, context, runtime, evals, observability, and deterministic execution.

The future is not “one big model with tools.”
The future is narrow capabilities, specialized agents, deterministic substrates, and management structures encoded into architecture.

Coding is becoming easy.
Knowing what should exist is not.

That was the day.
